# Article Interviewer - 使用例

## アプリケーションの実行

```bash
$ python main.py
```

## インタビューの流れ

```
Article Interviewer へようこそ！
記事作成のためのインタビューを開始します。

=== 記事作成インタビュー開始 ===

インタビュアーが記事について質問します。
各質問に回答してください。

質問 1/5: 記事のタイトルは何ですか？
回答: Pythonで始める機械学習入門

質問 2/5: この記事のメインテーマは何ですか？
回答: Pythonと機械学習ライブラリを使った基礎的な学習方法

質問 3/5: 想定読者はどのような人ですか？
回答: Pythonの基礎知識がある初心者プログラマー

質問 4/5: 記事で伝えたい主なポイントは何ですか？（複数ある場合はカンマ区切りで入力）
回答: 機械学習の基本概念, scikit-learnの使い方, 実践的なサンプルコード

質問 5/5: この記事を読んだ後、読者にどのような行動を取ってほしいですか？
回答: 実際に自分でコードを書いて機械学習モデルを作成してみる

=== インタビュー完了 ===

✓ インタビュー結果を草案として保存しました: drafts/draft_20251126_040154_Pythonで始める機械学習入門.md

草案ファイルを確認してください。
```
