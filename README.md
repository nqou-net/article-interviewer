# Article Interviewer

記事作成のためのインタビューツール

## 概要

Article Interviewerは、新しい記事を作成する際にインタビュアーのエキスパートと対話を行い、その結果を草案として保存するツールです。

## 機能

- インタビュー形式での記事内容のヒアリング
- エキスパートによる質問と回答の記録
- インタビュー結果の草案として保存

## セットアップ

```bash
pip install -r requirements.txt
```

## 使い方

```bash
python main.py
```

インタビューが開始され、記事のトピックや内容について質問されます。
すべての質問に回答すると、結果が草案として `drafts/` ディレクトリに保存されます。

## ファイル構成

- `main.py` - メインアプリケーション
- `interviewer.py` - インタビュアーモジュール
- `draft_manager.py` - 草案管理モジュール
- `drafts/` - 草案保存ディレクトリ
